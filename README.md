

🛡️ SENTINEL| Security System

SENTINEL  es un sistema inteligente de verificación digital diseñado para el
control de acceso en instalaciones universitarias. Este proyecto nace de la
necesidad de transformar la seguridad convencional en un modelo escalable basado
en biometría e identidad digital, con potencial de convertirse en una startup de
seguridad.

🚀 Características Principales

  - Verificación Inteligente: Identificación de Estudiantes, Profesores y
    Personal Administrativo.
  - Biometría Integrada: Interfaz preparada para reconocimiento facial y escaneo
    de huellas dactilares.
  - Control por QR: Generación y lectura de códigos de acceso únicos vinculados
    a la base de datos.
  - Base de Datos Local: Registro de usuarios con información detallada de
    carrera y rol.
  - Diseño Premium: Interfaz futurista "Glassmorphism" optimizada para
    dispositivos móviles.

🛠️ Tecnologías Usadas

  - Frontend: HTML5, CSS3 (Custom Variables & Animations), JavaScript (ES6+).
  - Librerías:
      - Html5-QRCode para el escaneo de cámaras.
      - QRCode.js para la generación de códigos.
      - Face-API.js (Estructura base para biometría facial).

📱 Instalación y Uso

1.  Clonar el repositorio:
    git clone https://github.com/tu-usuario/sentinel-ai.git
2.  Ejecutar: Simplemente abre index.html en cualquier navegador moderno o
    súbelo a GitHub Pages.
3.  Flujo de Usuario:
      - Registra un nuevo estudiante en la pestaña Registro.
      - Vincula biometría (Rostro/Huella).
      - Usa el Escáner para validar el acceso en tiempo real.

💡 Visión de Startup

Este sistema está diseñado para ser escalable. La arquitectura permite integrar
sensores de hardware externos, drones de vigilancia o cámaras de seguridad IP,
convirtiendo un simple registro de entrada en una red de seguridad inteligente
para empresas y campus.

Desarrollado como proyecto de innovación para la Facultad de Ciencias (LUZ FEC).
